# Ark Storage

> Simple file hosting for quickly sharing things with others!

## Features

-   Image Paste
-   Drag 'n Drop Files to Upload
-   Upload Queue / Jobs
-   Simple in-Browser File Editing (Image Cropping)
-   Public File Sharing URLs
